= Test::Unit::Runner::Tk

* http://rubyforge.org/projects/test-unit/

== DESCRIPTION:

Test::Unit::Runner::Tk - A Test::Unit UI built on Ruby/Tk.

== FEATURES/PROBLEMS:

* This provides "--runner=tk" option to use Tk UI.

== INSTALL:

* sudo gem install test-unit-runner-tk

== USAGE:

  require 'test/unit/runner/tk'
